## Contexto

Projeto: c:\Users\euron\Documents\guardia-de-choque

**Estratégia de resolução de tenant para a storefront:**
- A storefront é pública (sem auth) — usa `createServiceSupabase()` (service role) para buscar config
- O tenant é identificado pela env var `STORE_SLUG` (default: `guardia-de-choque`)
- Esta abordagem é segura: service role só é usado server-side, nunca exposto ao client
- Fase futura: resolução dinâmica por domínio (para multi-tenant)

**Arquitetura:**
- `layout.tsx` (Server Component) → chama `getStoreConfig()` → injeta result no `<TenantProvider>`
- `<TenantProvider>` (Client Component) → expõe via React Context
- `useStoreConfig()` → qualquer client component lê a config

## TASK 1 — Criar src/lib/store-config.ts

```typescript
import { createServiceSupabase } from '@/lib/supabase-server';
import type { TenantConfig } from '@/types/tenant';

const DEFAULT_CONFIG: TenantConfig = {
  id: '',
  tenant_id: '',
  brand_name: 'Minha Loja',
  logo_url: null,
  primary_color: '#0F172A',
  accent_color: '#059669',
  font_heading: 'Playfair Display',
  font_body: 'DM Sans',
  phone: null,
  email: null,
  domain_display: null,
  announcement_messages: [],
  trust_messages: [],
  faq_items: [],
  stats: [],
  seo_title: 'Minha Loja',
  seo_description: null,
  og_image_url: null,
  updated_at: new Date().toISOString(),
};

/**
 * Busca a configuração pública da loja pelo slug do tenant.
 * Usa service role (server-only) para bypass de RLS.
 * Retorna config com defaults se não encontrar.
 *
 * @param slug - slug do tenant (default: STORE_SLUG env var)
 */
export async function getStoreConfig(slug?: string): Promise<TenantConfig> {
  const tenantSlug = slug ?? process.env.STORE_SLUG ?? 'guardia-de-choque';

  try {
    const supabase = createServiceSupabase();

    const { data: tenant, error: tenantError } = await supabase
      .from('tenants')
      .select('id')
      .eq('slug', tenantSlug)
      .single();

    if (tenantError || !tenant) return DEFAULT_CONFIG;

    const { data: config, error: configError } = await supabase
      .from('tenant_config')
      .select('*')
      .eq('tenant_id', tenant.id)
      .single();

    if (configError || !config) return DEFAULT_CONFIG;

    return config as TenantConfig;
  } catch {
    return DEFAULT_CONFIG;
  }
}
```

## TASK 2 — Criar src/components/providers/TenantProvider.tsx

Criar o diretório `src/components/providers/` (se não existir) e o arquivo:

```typescript
'use client';

import { createContext, useContext } from 'react';
import type { TenantConfig } from '@/types/tenant';

const TenantContext = createContext<TenantConfig | null>(null);

export function TenantProvider({
  config,
  children,
}: {
  config: TenantConfig;
  children: React.ReactNode;
}) {
  return (
    <TenantContext.Provider value={config}>
      {children}
    </TenantContext.Provider>
  );
}

/**
 * Hook para acessar a configuração da loja em qualquer Client Component.
 * Deve ser usado dentro de <TenantProvider>.
 */
export function useStoreConfig(): TenantConfig {
  const ctx = useContext(TenantContext);
  if (!ctx) {
    throw new Error('useStoreConfig deve ser usado dentro de <TenantProvider>');
  }
  return ctx;
}
```
